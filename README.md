## Proyecto
Rescue the kingdom
Un juego 2D en el que controlaremos a un caballero que se encuentra en un castillo con el objetivo de rescatar el reino del dominio del Black Knight.

El juego tendrá una perspectiva de vista superior (Top-Down).

Uso del A* en el juego:
El algoritmo A* se utilizará para que los enemigos detecten al jugador y calculen la ruta más corta para alcanzarlo y atacarlo.
También se empleará para evitar obstáculos dentro del mapa.

Uso del Árbol de Conocimiento:
Mediante el árbol de conocimiento, los enemigos o NPC decidirán qué acción realizar dependiendo de la situación, como perseguir o atacar
